About Demo Mode


The purpose of demo mode is to allow recruitors to see first-hand how the application works while protecting student information under Gideon. The app works similarly in demo mode as it does with authorized users, but with a few key limitations:

  • The sample data displayed is minimal in nature and final.
  • Student and record data cannot be inserted, updated, or removed in the database, although the forms can still be used.
  • The application's "present date" is set to a constant March 1st, 2019.
  • International goal lines on progress charts are empty and will never appear. (for now)
